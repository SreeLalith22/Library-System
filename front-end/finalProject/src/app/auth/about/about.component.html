About Us

This is a Library System, which creates a virtual library-like environment, where users can search and borrow books of their choice. This design was implemented using Angular framework with TypeScript language for the front end and pure TypeScript for the back end with REST API. As a team of three, we are honored and proud to have been able to develop a full web application, and meet the necessary criteria for this project at Maharishi International University, and receive great guidance from Professor Asaad.
